Diagnostic logging is now a toggle (Settings → Advanced), off by default.

Previously cairn continuously captured a rolling log file across launches on every device — useful for bug reports, but needless background work (a poll every 20s + disk writes) for everyone else. Now it's opt-in:

- Settings → Advanced → "Diagnostic logging." Off by default.
- To send a useful bug report: turn it on, reproduce the issue, then "Export diagnostic logs" (right below the toggle).
- Export still works while the toggle is off — it just covers the current session rather than history across launches.

Existing installs default to off after this update.

No other changes in this build.
